Usage:
  lexgen [ opt [opt-value] | <key>=<value> ]...

Options:
  -h --help                        Show this message.
  -i --in                          Input file path.
  -o <path> --out <path>           Output file path.
  --out-inc <path>                 Output include file path. This option works
                                   only when the language needs header file,
                                   such as C/C++.
  --table-only                     Only generate the transition table.
  -l=<value> --lang[uage]=<value>  Specify language.
    c                              This is the default value.
    cpp
    rust                           Currently not supported.
