Usage:
  pargen [ opt [opt-value] | <key>=<value> ]...

Options:
  -h | --help                       Show this message.
  -i | --in                         Input file path.
  -o <path> | --out <path>          Output file path.
  --out-inc <path>                  Output header file path. This option works
                                    only when the language needs a header file,
                                    such as C/C++.
  -t <path> | --tmpl <path>         Specify template file for generating source
                                    file.
  --it <path> | --inc-tmpl <path>   Specify template file for generating header
                                    file.
  --li <prefix> | --lrinfo <prefix> Output LR table information to files. Files
                                    will be output to a directory, which is
                                    specified by <prefix>.
  -l=<value> | --lang[uage]=<value> Specify language.
    c                               This is the default value.
    cpp
    others                          Use --ls-lang to check supported languages.
  --ll | --ls-lang[uage]            List all supported languages.
